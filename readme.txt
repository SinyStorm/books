На сервере должна быть установлена Java 8 и postgres.
books.sql - дамп базы со справочниками, users.sql - дамп пользователей.

Для запуска:
1. Положить собранный jar-ник и start.sh в какую-нибудь папку на сервере и перейти в нее.
2. Выполнить команду pkill -9 java (остановить ранее запущенное приложение)
3. Выполнить команду start: nohup ./start.sh > log.txt 2>&1 & (запустит приложение)
